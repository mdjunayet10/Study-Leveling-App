Study Leveling is prepared for:

- Web
- Android
- iOS
- Linux

- Study task system
- XP and coin rewards
- Leveling system

- Progress tracking
- Global leaderboard
- Home multiplayer mode
- Away multiplayer room support
- Account/login support
- Web hosting support
