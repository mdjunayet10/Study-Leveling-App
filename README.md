# Study Leveling App

**Study Leveling** is a gamified study productivity app designed to help students stay motivated, complete study tasks, track progress, earn rewards, and study together.

This public repository is used for the official app page, release notes, license, screenshots, and public downloads.

The full application source code is private.

---

## Live App

Firebase Hosting:

```text
https://study-leveling.web.app
```

---

## Platform Support

Study Leveling is prepared for all major platforms:

- Web
- Android
- iOS
- Linux
- macOS
- Windows

---

## Main Features

- Study task system
- XP and coin rewards
- Level-up progress
- Halal-friendly reward shop
- Global leaderboard
- Home multiplayer mode
- Away multiplayer study rooms
- Login and account support
- Firebase-powered leaderboard and multiplayer features
- Cross-platform app support
